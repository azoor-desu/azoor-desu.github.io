Logs a message. Message will be formatted and printed to console, written to log file, and sent to editor (if any). Not to be used directly, use TE_DEBUG or its variants.

Parameters
logLevelThe level of logging to tag this message as (Trace, Fatal, Warn, etc)
useMsgSystemSend this message to the messaging system?
formatThe format of the message and the parameters after it. Works like printf.
