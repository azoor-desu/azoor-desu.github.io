Turns an entity into a prefab source, adds it into the internal prefab source storage on the engine, and also writes the prefab source to disk. If existingPrefabSourceGUID is specified, entity's current values, components and overrides will be written into the existing Prefab Source instead of creating a new one. If name of a prefab source matches this new prefab source, this new prefab source's file anme will have a number appended to file name, but the entity name will not be modified.

Parameters
entityentity to create prefab source from. Entity will not become prefab.
existingPrefabSourceGUIDOptional. Specify to overwrite a prefab source.
