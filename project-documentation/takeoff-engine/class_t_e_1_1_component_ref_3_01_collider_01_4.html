Detailed Description

Specialization for ComponentRef<Collider>. ECS does not store a container of COlliders, but rather, containers of individual collider types (Box, Sphere etc.) Collider is NOT registered in the ECS (and should not be), but need to use ComponentRef<Collider>. Leaving the generation of the ComponentRef<Collider> template to the generic one will return the wrong object type.
