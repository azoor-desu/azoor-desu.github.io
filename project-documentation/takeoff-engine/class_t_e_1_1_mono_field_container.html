void * TE::MonoFieldContainer::GetValue ( _MonoObject *  objectInstance) const

Returns the pointer to the memory location in Mono where this data is stored. There is no gurantee that the data will be in the same location after awhile. If underlying value is an array or string, a MonoString* or MonoArray* type will be returned instead. If underlying type is a struct, MonoObject* will be returned. mono_object_unbox needs to be used to unbox this object. Custom C# structs like Vec3 will be considered as a MonoObject*. C# Classes have different memory layout so classes may not work properly. https://github.com/mono/mono/issues/12608.
