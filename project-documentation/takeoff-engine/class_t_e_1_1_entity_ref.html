Detailed Description

Holds a reference to an entity. Does not directly contain data of the entity. To get an entity, either use ecsManager.CreateEntity() or ecsManager.FindEntity(). Creating EntityRef by itself without assigning it to anything will create a null reference entity. entity1 = entity2 will result in both Entity Objects referring the the same entity. Access this entity's data by using the various Getter and Setter functions. This class can be passed around as a copy, as the internals are very lightweight.
