◆ InvokeMethod()

MonoObject * TE::ScriptManager::InvokeMethod ( const MonoMethodContainer &  monoMethod,
_MonoObject *  objectInstance,
void **  params = nullptr 
)
static

Uses the MonoMethod provided to invoke a method in the C# managed code. If the target method is static, objectInstance should be null. If the monoMethod has a unmanaged thunk version, the function will call that instead.

Parameters
monoMethodThe MonoMethod container taht contains the method to be called.
objectInstanceThe object instance that the method should belong to.
paramsAny additional parameters to pass to the method to be called. If unmanaged thunk is used, this will be ignored.
Returns
Returns a mono object. the mono method could return anything, so it's wrapped in a MonoObject.
