bool TE::SphereCollider::CollidingMoving ( SphereCollider *  other,
glm::vec3  d,
TEfloat  t0,
TEfloat  t1,
TEfloat &  timeOfCollision 
)

For a moving sphere that may tunnel through another sphere, we need to find if the moving sphere is going to intersect with the other soon. We can do this by using an interval halving method. Creating a bounding sphere encapsulating the moving sphere's path over time, we can check if the bounding sphere is colliding with the other sphere.

If it is, half the interval & check again. Repeat this until the interval is small enough to be considered a collision or if the smaller bound does not intersect with the other sphere. Static checks are done if the big bounding sphere is colliding with the other sphere. Spheres intersect if squared distance is less than squared sum of radii.

Parameters
otherthe other sphere to check collision with.
dthe direction the sphere is moving in.
t0the start of the interval.
t1the end of the interval.
timeOfCollisionoutput paramater of the time of collision.
