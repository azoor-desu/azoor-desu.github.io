Detailed Description

Main interface for ECS related things

1. Registration of Components/Systems, getting handles to these guys and management of these guys.

2. Creation/deletion of Entities and Entity related manipulation (Cloning, copying)

3. Getting component data (e.g. array of components) for Systems to use

4, Standard Manager things (initmanager etc)
