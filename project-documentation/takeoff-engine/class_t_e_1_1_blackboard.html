Detailed Description

Allows systems to "post" events onto the blackboard and let other systems "read" events posted on the blackboard. 2 types of posts:

  1. Timed post, where the poster posts an event that lasts for a certain amount of time e.g. 2 seconds or 1 frame. Poster can choose if the post can be removed by others, or cannot be removed until the time runs out.
  2. Untimed post. Post remains on the blackboard until another system takes it down.

Feels like a knockoff messaging system... but messaging system has bugs and is too specialized for some uses.
