 Gets a random and unique type GUID that is generated during compile time. Getting TypeGUID of the same type/class will always return the same value. As long as the NAME of the type is different, this variable will return a new type GUID. e.g. TEguid and TEtypeguid are both uint64, however they will return DIFFERENT TypeGUIDs.
