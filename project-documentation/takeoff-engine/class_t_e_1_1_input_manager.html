Callback function to be called when GLFW window receives a keyboard input(keys pressed, held and released)

Parameters
windowThe pointer to the GLFW window.
keyThe KeyCode for specific button of the event that triggered the callback.
scancodeThe scancode for specific button of the event that triggered the callback, UNUSED BY INPUTMANAGER
actionThe action (released = 0, pressed = 1, repeat =2) for specific button of the event that triggered the callback.
modsThe mods for specific button of the event that triggered the callback, UNUSED BY INPUTMANAGER
