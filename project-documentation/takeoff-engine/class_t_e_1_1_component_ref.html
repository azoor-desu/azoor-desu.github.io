Detailed Description

template<typename ComponentType>
class TE::ComponentRef< ComponentType >

Holds a reference to a component. Does not directly contain data of the component. To access the underlying component, use the > operator Creating this class by itself without assigning it to anything will create a null reference entity. ref1 ref2 will result in both ref Objects referring the the same component. This class can be passed around as a copy, as the internals are very lightweight.
