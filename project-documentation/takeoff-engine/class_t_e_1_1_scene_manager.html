◆ SaveCurrentSceneAsNewScene()

void TE::SceneManager::SaveCurrentSceneAsNewScene ( TEstringview  newSceneName,
TEstringview  fullPath = nullptr 
)

Only for editor use. Saves the current scene as a new scene, leaving the original scene unmodified. Will save the new file to the same folder that this current scene is in, under a new name provided by the user. If the name matches any existing files, a number will be appended to the new file name to avoid overwriting existing files. If the name matches any existing scenes internally (but not saved on disk), a number will be appended to the new file name to avoid overwriting existing files.

Parameters
newSceneNameName of the new scene
fullPathOptional Full path to save this new file into. If this option is provided, user may save the file to any desired path on windows. No checks for file conflict will be done, and the newly created scene will not be added into the internal list if it is out of the Assets folder.
